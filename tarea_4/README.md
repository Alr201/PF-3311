# Tarea 4 - AmigoGPT
Esta tarea consiste en hacer un chatbot básico utilizando la API de Gemini. El chatbot está configurado para responder como un amigo cercado, utilizando un tono relajado, empático y natural.

## Instalación
1. Clonar el repositorio: 
   ´´´
   git clone https://github.com/Alr201/PF-3311.git 
   ´´´
2. Instalar dependencias:
   ´´´
   pip install google-generativeai python-dotenv
   ´´´
## Configuración 
Crear un archivo ´.env´ y poner la API Key ahí.

## Uso

Ejecutar el programa:
´´´
python main.py
´´´
